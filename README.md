# Agentic Engineering From Scratch

## Philosophy: Build to Understand

Production-grade AI Engineering isn't about gluing together pre-built components; it's about owning the cognitive architecture of your agent. High-level frameworks are excellent for rapid prototyping, but they obscure the internal mechanics required for building robust, reliable, and observable systems.

This repository follows a "Build-to-Understand" philosophy. We use only Vanilla Python and Pydantic to construct every critical piece of an agentic system from scratch: the prompting layer, the RAG pipeline, the core orchestration loop, and the observability stack. By building the engine, you will gain the deep, systems-level understanding necessary to debug, evaluate, and scale agentic systems for real-world applications. This is not the fastest path, but it is the only path to true mastery.

## Curriculum

This repository is divided into four modules that build on each other, taking you from the fundamentals of language model interaction to building a production-grade, observable agent.

*   **Module 01: The Prompt Engineering Layer:** Learn to steer the "Brain" of the agent.
*   **Module 02: Embeddings & Retrieval-Augmented Generation (RAG):** Ground the model in factual, up-to-date data.
*   **Module 03: Core Agent Anatomy:** Deconstruct an agent into its functional systems.
*   **Module 04: Agent Quality & Observability:** Transition from proofs-of-concept to production-grade reliability.
